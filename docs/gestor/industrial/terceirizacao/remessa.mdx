## Remessa para tercerização
### Precisa
- OP
- Tercerizada/parceiro
- Processo
- Produto
> Produto não pode ter sido já apontado como defeituoso pois isso o inativa, sendo impossível utilizá-lo de qualquer forma após esse apontamento. Somente após retorno que é informado o defeituoso realmente que irá ser descartado. Processo deve ser atrelado ao setor que foi feito o apontamento, que seria o local físico que está o produto para ser enviado para tercerização.
