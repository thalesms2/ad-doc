# Atalhos
## pgAdmin III
Tecla               | Função
------------------- | -----------------------------------------------------------
Ctrl + E            | Com o banco selecionado abre o executador de SQL Querys
F5                  | Executa o código 
Ctrl + K            | Comenta a linha selecionada
Ctrl + Shift + K    | Descomenta a linha selecionada
Ctrl + U            | Converte o texto selecionado em letras maiúsculas
Ctrl + Shift + U    | Converte o texto selecionado em letras minúsculas
Ctrl + D            | Abre as informações da tabela selecionada
Ctrl + G            | Filtro
