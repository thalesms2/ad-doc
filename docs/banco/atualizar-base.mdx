# Atualizar base de dados interna
*SOMENTE NO BANCO INTERNO*
É necessário atualizar a base dados para validar se o problema em questão
também ocorre na versão mais atual, para ter certeza que o problema ainda não foi
solucionado pelo desenvolvimento.

Dentro da central de downloads na pasta da atualização possui a pasta `Script/Geral`
É necessário pegar os scripts e rodar em ordem.
Por exemplo: `22.1 > 22.2 > 22.3 > 22.4 > 23.1`.
