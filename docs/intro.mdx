# Padrões
Manuais e documentações referentes à sistemas oferecidos pela [Adsomos.](https://adsomos.com/)
## Padrão
Favor ao editar ou adicionar novos tópicos, utilizar o seguinte padrão:
### Nome do arquivo
Nome deve ser curto, descrevendo o conteúdo utilizando siglas, deve ser separado por `-`.
### Ao adicionar um arquivo novo
Sempre ao adicionar um arquivo novo, deve-se ao criá-lo, adicioná-lo 
no `meta.json` do tópico que o envolve, utilizando tanto o nome completo quanto as siglas, 
exemplo `Nota fiscal eletrônica NF-e`. O tópico deve ser escolhido pelas seguintes regras:
- Módulo gestor:
    - Onde o procedimento é feito;
    - Onde o erro acontece;
    - Dúvida ou informação sobre algo do módulo.
- Por sistema, exemplo: `SPED Integrado`, `TNFe` e afins.
- Suporte englobando telefonia, sistema do suporte (Movidesk).
### Checklist de itens dentro do arquivo
- [ ] Deve possuir título;
- [ ] Deve possuir o caminho que é feito;
- [ ] Deve possuir imagens sempre que possivel;
### Caso não for possivel por algum motivo não seguir as regras à cima
Insira o arquivo na pasta de `homologação` e insira no meta com qualquer nome, 
que será validado para entrar nos padrões e inserido no local correto.
