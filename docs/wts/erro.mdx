# Servidor WTS com tela preta
Usuário do servidor WTS está travado, para solucionar é necessário logar no servidor
com privilégios de administrador, acessar o gerenciador de tarefas `Ctrl + Shift + Esc` 
selecionar a aba `Usuários`, selecionar o usuário travado, clicar com lado direito e 
desconectar. Isso fará com que o usuário seja deslogado forçadamente pelo servidor.
